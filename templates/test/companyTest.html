Status

200 : API 호출에 성공하였고 결과값이 존재하는 경우
201 : API 호출에 성공하였으나 결과값이 존재하지 않는 경우
400 : DB 조회할 때 에러가 난 경우
401 : 입력값에서 디비 또는 API 파라미터에 입력할 수 없는 특수문자가 포함되어 있는 경우

CURRENTPAGE : 0 => 1페이지

Status

200 : 정상작동
400 : 업체명 중복
401 : 업체명 크기 부적합(1~100자)
402 : 업체명 타입 부적합(str)
403 : 주소 크기 부적합(~100자)
404 : 주소 타입 부적합(str)
405 : 전화번호 크기 부적합(12~13자)
406 : 전화번호 타입 부적합(str)
407 : 이메일 크기 부적합(~100자)
408 : 이메일 타입 부적합(str)
409 : 메모 크기 부적합(~255자)
410 : 메모 타입 부적합(str)
411 : 하루 최대 발급 개수 타입 부적합(int)
412 : 최대 발급 개수 타입 부적합(int)
413 : 최대 발급 날짜 타입 부적합(str)
414 : 업체를 저장하는 도중 DB 조회할 때 에러가 난 경우
415 : 업체를 저장하고 저장한 업체 리스트를 불러오는 곳에서 DB 조회할 때 에러가 난 경우
416 : 입력값에서 디비 또는 API 파라미터에 입력할 수 없는 특수문자가 포함되어 있는 경우

입력값

USER KEY : str 타입의 8자리 숫자가 아닌 경우 회원가입에서 업체정보 자가 등록으로 인식 (ex. 0 입력 시 자가 등록)
